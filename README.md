# WilDev.AutoSave
Unturned plugin to add automatic saving through the OpenMod API

## How to Install
Make sure you are in-game and run this command:
`/openmod install WilDev.AutoSave`

## Documentation
*Config.yaml*
```yaml
Save-Server: true # If the server should be saved - Must be a boolean value
Save-Interval: 90 # Interval between saves in seconds - Must be a boolean value
Announce-Saves: true # If saves should be announced - Must be a boolean value
Announce-URL: images.unturnedserver.net/images/image.png # Announcement URL for the image when announcements are sent - Must be a string value without double quotes
```
*Translations.yaml*
```yaml
Saving: "Saving server..." # Must be a string value - Useable Parameters: Rich Text <>
```
*Parameters*
- Rich Text <>: To add color and text formatting support for in-game text - Must be configured with <>
